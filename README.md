## درباره استیو وی‌پی‌ان

این پروژه با تمرکز بر تجربه کاربری روان، پایداری اتصال و مدیریت ساده سرورها و اشتراک‌ها ساخته شده است. رابط کاربری اپ با افکت‌های Liquid Glass طراحی شده تا حس مدرن و شفافی به کاربر بدهد.

### ویژگی‌های کلی

- Xray و OpenVPN پشتیبانی از هسته 
- مدیریت سرورها و اشتراک‌ها با کش هوشمند و بروزرسانی خودکار
- صفحه تنظیمات اختصاصی برای پیکربندی دقیق اتصال
- بررسی خودکار آپدیت و اطلاع‌رسانی نسخه جدید
- طراحی مدرن با افکت‌های شیشه‌ای (Liquid Glass)
- عملکرد پایدار و بهینه‌شده برای مصرف کمتر منابع
